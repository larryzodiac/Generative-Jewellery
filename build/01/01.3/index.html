Octahedron Geometry Subdivision

This exercise is a demo of Catmull-Clark Subdivision surfaces implemented in Three.js using half-edge data structure principles. The maths is off quite a bit but it's getting there.

Press ↑ or ↓ to subdivide. Drag the mouse to rotate the camera around the shape. Scroll to zoom.

See half-edge, face, vertex node and vertex list on threejs.org.
