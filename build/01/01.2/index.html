Cone Geometry Subdivision

Three.js uses loop subdivision, an algorithm defined for for triangular meshes.

This exercise investigates the SubdivisionModifier.js file so to study how subdivision formulae are implemented using Three.js' principles.

Press ↑ or ↓ to subdivide. Drag the mouse to rotate the camera around the cone. Scroll to zoom.

See subdivision modifier on threejs.org.
